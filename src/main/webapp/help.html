Генерирование справки:

GET: /getGeneratedWordDocument

Обязательные параметры:

    requestType* =
  • 2 - Справка о социальной помощи
  • 4 - Справка о получении социальной помощи (не препятствовать проезду)
  • 6 - Направление на санитарную обработку
  • 8 - Направление на диспансеризацию
  • 10 - Справка о регистрации
  • 12 - Справка по "Транзиту"
  • 14 - Запрос в отдел ЗАГС
  • 16 - Произвольная справка
    clientId* - ID клиента, для которого формируем справку

Необязательные параметры:

    issueDate - дата формирования справки (формат дд.мм.гггг). Если не указана, то используется текущая.

Параметры, относящиеся к конкретным запросам:

    travelCity - город в который направляется клиент (тип 4 и тип 12)

Пример: http://HOST:PORT/homeless-report-engine/getGeneratedWordDocument?requestType=2&clientId=13000&issueDate=09.09.2009&workerId=2

Генерирование контракта (в случае наличия, новый контракт не генерируется, отдается уже существующий)

GET: /getGeneratedContract

Обязательные параметры:

    requestType* =
  • 100 - Стандартный контракт
  • 102 - Контракт с проживающим
    contractId* - ID контракта для формирования документа (у одного клиента их может быть несколько)
    clientId* - ID клиента, для которого формируем документ
    workerId* - ID работника, формирующего документ

Необязательные параметры:

    issueDate - дата формирования контракта (формат дд.мм.гггг). Если не указана, то используется текущая.

Параметры, относящиеся к конкретным запросам:

Пример: http://HOST:PORT/homeless-report-engine/getGeneratedContract?requestType=100&clientId=13000&contractId=328&issueDate=10.10.2010&workerId=2
